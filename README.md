# Dr. Liuba Garaga — Dermatolog

Modern bilingual (Romanian / English) website for Dr. Liuba Garaga, dermatology specialist in Toplița, Romania.

## Sections

- **About** — Personal introduction and professional background
- **Services** — Full list of dermatological and aesthetic services offered
- **Booking** — Appointment information (online booking coming soon)
- **Contact** — Location (Google Maps), phone, email, and contact form

## Features

- 🌐 Romanian / English language toggle
- 📱 Fully responsive design
- ✨ Modern, clean UI with smooth animations
- 🗺️ Embedded Google Maps location

## Getting Started

Simply open `index.html` in a web browser. No build tools or dependencies required.

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Inter)
